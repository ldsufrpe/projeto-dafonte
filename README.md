# Projeto Água

Página HTML simples. Este repositório contém os arquivos do projeto para publicação como página estática.

Como publicar no GitHub:

1. Criar repositório público no GitHub.
2. Fazer push do conteúdo para a branch `main`.
3. Nas configurações do repositório (Settings) > Pages, selecionar branch `main` e pasta `/` ou `/docs`.
